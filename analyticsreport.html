ONS Crime Data Report

Assignment Description

The task was to comprise an analytical report from any 24-month period of the ONS crime data. Specifically, to report on the refinement and integration of data, statistical dataset analysis using the measures of mean, mode, median, standard deviation, correlation, variance, and covariance; create a hypothesis definition and test it using the relevant test measures; implement a set of prediction models on the dataset using at least 2 different methods; analyse the effectiveness of the prediction models and discuss around other potential solutions; provide references to at least 7 journals from the last 5 years; and provide all code used in the appendix.

I gave a comprehensive explanation on the data cleaning, merging and feature creation that was required to be able to pull information from the data. Then used a variety of visualisation techniques from the seaborn library to show findings and trends with potential reasons and support from journals. Discussed various issues and solutions that surrounded the usage of data for machine learning and implemented a linear regression, SARIMAX, and classification model.

Through the feedback given by the module leader, the report was said to be excellent and meets all the expected requirements. The descriptive analytics was correct and critical with a comprehensive comparison of prediction techniques.
