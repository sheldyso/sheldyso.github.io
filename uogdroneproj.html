UOG Drone-Ing On Project

About the Project

My role within the team was a programmer. My initial responsibilities were to program a script which would track an objects movement in unity and store it in a database. The script was programmed using C# and the SQLite .NET package. 2 other team members produced a to-scale size 3D model of the room and to-scale size drone. The initial training was done in Unity where Navmeshes were used to path-find between 2 points, avoiding pillars in the room. The script tracked the x and y position and updated the database each second.

Following this, I programmed a python script which used the drones API and built a class which controls the drone. The drone used was a DJI educational-series drone. The class takes all positions from the database and allows the user to tell the drone to move to the next position or previous using the arrow keys. It was entirely possible to allow the drone to move of its own accord but an element of control needed to be maintained at all times until obstacle avoidance and detection measures were implemented.

During a test, the drone was able to move from one end of the room to the other but did require some manual adjustments in the data. The next step from this is to refine the shape and size of the room model and undergo some more path-finding and store the data. Once we are confident that the sizes are accurate, then there may be considerations on allowing the drone to be fully autonomous with a manual override.
